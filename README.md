### Installation

#### Linux (Ready Now)

```bash
# AppImage (universal)
chmod +x "Aethra Browser-1.0.0.AppImage"
./Aethra\ Browser-1.0.0.AppImage

# Debian/Ubuntu
sudo dpkg -i aethra-browser_1.0.0_amd64.deb

# Snap
sudo snap install aethra-browser_1.0.0_amd64.snap --dangerous
```

#### Windows (5 minutes)

1. Download build-windows.bat
2. Run on Windows PC
3. Install generated .exe

#### macOS (5 minutes)

1. Download build-macos.sh
2. Run on Mac: `bash build-macos.sh`
3. Install generated .dmg

#### Android (30 minutes)

1. Download android/ folder
2. Follow README.md
3. Build APK with React Native
